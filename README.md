# Playwright E2E Framework

## Setup
- npm install
- npx playwright install

## Run MCP for AI generation
- npm run mcp

## Run tests
- npx playwright test

## Estructura
- tests/: Tests E2E
- pages/: Page objects
- fixtures/: Fixtures
- utils/: Helpers
- prompts/: Prompts usados con IA
- docs/: Documentación (ai-review.md)

Evidencia de IA: Usé GitHub Copilot en VS Code para generar código inicial basado en prompts en /prompts.
